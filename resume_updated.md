# 简历内容（已按项目代码核对修订，字数格式大体不变）

个性化学习辅导 Agent 系统 | 大模型应用开发 | 2026.05 - 至今

- 基于 LangGraph 构建 Router+4 Agent 架构，自动识别意图并动态路由；各 Agent 配备代码执行/习题生成工具，路由准确率达 80.8%（实测）。
- 针对课程内容检索痛点，引入 RAG 优化方案：基于 QwenEmbeddings 实现语义分块(800+150)，混合检索(向量+关键词)显著提升长尾知识点检索命中率。
- 封装 MCP 工具标准化接入（MultiServerMCPClient），新工具接入周期从 2 天缩短至 2 小时，支持动态扩缩容。
- 设计三层记忆架构：Redis 短期记忆 + 业务逻辑中期记忆 + ChromaDB 知识图谱长期记忆，实现个性化学习路径。
- 采用 FastAPI + SSE 实现流式交互，结合 D3.js 可视化知识图谱，提升学习体验与参与度。
